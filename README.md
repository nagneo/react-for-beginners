# MVING - 영화 웹서비스 만들기

### 개요
* 2022년 react를 공부하면서 처음 만들었던 프로젝트 입니다. 
* SPA로 만들어져 있으며, [open api](https://api.themoviedb.org/)를 사용하여 영화 관련 정보를 보여줍니다.
* vecel르 배포 하였습니다. [Visit](https://mving.vercel.app/)

### 환경

#### 사용 기술 스택


* react v18.2.0
* swiper v2.1.4
* emotion v11.11.3

#### 개발 환경

* node 18.18.0 (npm 8.19.4)

### local 실행

```
# npm 설치
npm install

# local 실행
npm run dev

```
